# Dischord

MML (Music Macro Language) Bot for Discord

## Description

Dischord is the bot that allows you to compose songs with MML.

## Requirement

- Node.js

- FFmpeg

## Usage

### Run Dischord

1. Run `npm install`

1. Run `npm run start`

1. Yes! Party Time!!

### Bot Usage

Check out usage by sending `dc!help` to Dischord.
